# 🎤 AISpeechGenerator

API для генерации структурированных речей с использованием Microsoft Phi-3-mini.

## 🚀 Функциональность

### Основные функции
- **Генерация структурированных речей** с вступлением, основной частью и заключением
- **Поддержка различных стилей выступлений** (научный, вдохновляющий, разговорный и др.)
- **Настройка параметров речи**: тема, длительность, ключевые моменты
- **Управление стилями** через REST API (добавление, получение, обновление)
- **Динамическая настройка параметров модели** генерации текста
- **Поддержка русского языка** для входных данных и выходных речей

### Особенности
- **Асинхронная архитектура** на FastAPI для высокой производительности
- **Dependency Injection** для управления зависимостями
- **Singleton паттерн** для единого экземпляра модели
- **Валидация данных** с помощью Pydantic

## 🛠 Технологии

### Основной стек
- **Python 3.8+** - основной язык программирования
- **FastAPI 0.121.3** - современный веб-фреймворк для создания API
- **Uvicorn 0.38.0** - ASGI-сервер для запуска приложения
- **PyTorch 2.9.1** - фреймворк для глубокого обучения
- **Transformers 4.57.1** - библиотека для работы с NLP моделями
- **Pydantic 2.12.4** - валидация и сериализация данных
- **Accelerate 1.12.0** - оптимизация работы с моделями на разных устройствах

### Модель генерации

- **Модель**: Microsoft Phi-3-mini-4k-instruct
- **Контекстное окно**: 4096 токенов
- **Тип модели**: Инструктивная (instruct-tuned)
- **Назначение**: Генерация текста на основе промптов

## 📦 Установка и настройка

### Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- 8+ GB оперативной памяти (рекомендуется)
- CUDA-совместимая видеокарта (опционально, для ускорения)

### Шаги установки

1. **Клонирование репозитория**
   ```bash
      git clone <repository-url>
      cd speech-generator

2. **Установка зависимостей**
   ```bash
      pip install -r requirements.txt

3. **Запуск сервера разработки**
   ```bash
      python main.py

4. **Настройка окружения**
   Создайте файл .env в корне проекта
   Заполните необходимые переменные:
      MODEL_NAME=microsoft/Phi-3-mini-4k-instruct
      DEVICE=cpu  # или cuda для GPU
      CACHE_DIR=./model_cache
      LOG_LEVEL=INFO
      PORT=8000

## 🎯 Использование

1. **Ручной запуск**
   ```bash
      python main.py

2. **Пример запроса на генерацию речи**
   ```bash
      curl  -X POST "//localhost:8000/generate-speech/"\
            -H "Content-Type: application/json" \
            -d '{
               "topic": "Искусственный интеллект в современном мире",
               "duration_minutes": 5,
               "style": "научный",
               "key_points": ["История ИИ", "Современные применения", "Этические вопросы"]
            }'

## 🔧 Настройка модели

### Параметры по умолчанию
   {
    "max_length": 500,
    "temperature": 0.7,
    "top_p": 0.9,
    "repetition_penalty": 1.1
   }

### Доступные API эндпоинты:
   POST /generate-speech/ - генерация речи  
   POST /set-model-settings/ - настройка параметров модели  
   GET /styles/ - получение списка стилей  
   POST /styles/ - создание нового стиля  
   PUT /styles/{style_id} - обновление стиля  
   GET /health/ - проверка работоспособности API  
   GET /model-info/ - информация о модели
